# fallow-agent

Idle detection, rapid preemption, process supervision, model-cache and batch-worker modules for
Fallow worker machines. The runnable daemon composition is not yet implemented.

Fallow is pre-alpha. See the [repository README](https://github.com/Unluckyathecking/fallow#readme),
[compatibility policy](https://github.com/Unluckyathecking/fallow/blob/main/docs/compatibility.md)
and [license](https://github.com/Unluckyathecking/fallow/blob/main/LICENSE).
