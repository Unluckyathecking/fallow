# fallow-coordinator

Registry, authentication, queue, scheduler, model-distribution and OpenAI-compatible gateway
modules for the Fallow coordinator. The runnable composition entrypoint is not yet implemented.

Fallow is pre-alpha. See the [repository README](https://github.com/Unluckyathecking/fallow#readme),
[roadmap](https://github.com/Unluckyathecking/fallow/blob/main/ROADMAP.md) and
[license](https://github.com/Unluckyathecking/fallow/blob/main/LICENSE).
