# fallow-cli

The `flw` operator command-line client for the Fallow admin API. Server-side admin API
composition is not yet implemented, so this package is currently for development only.

Fallow is pre-alpha. See the [repository README](https://github.com/Unluckyathecking/fallow#readme),
detailed [CLI documentation](https://github.com/Unluckyathecking/fallow/blob/main/packages/fallow-cli/src/fallow_cli/README.md)
and [license](https://github.com/Unluckyathecking/fallow/blob/main/LICENSE).
