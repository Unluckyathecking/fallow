# Support

Fallow is a pre-alpha, volunteer-maintained project. Community support is best effort and there
is no guaranteed response time.

Use GitHub Discussions for setup questions, design discussion and usage ideas when Discussions
is enabled. Otherwise, open a question-labelled issue. Search existing issues first.

For reproducible defects, use the bug report form and include:

- Fallow commit or release, Python and uv versions
- Operating system, CPU/GPU and relevant driver versions
- The command run and the smallest reproducible example
- Expected and actual behaviour, with sanitised logs
- Whether the problem reproduces without model or network dependencies

Remove credentials, prompts, documents, hostnames, IP addresses and other sensitive deployment
data. Security vulnerabilities must follow [SECURITY.md](SECURITY.md), not public support
channels.

Commercial support, deployment certification and production-readiness guarantees are not
currently offered.
